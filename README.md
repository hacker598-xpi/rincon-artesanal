# El Rincón Artesanal

Sitio web estático de venta de piezas de barro artesanales.
Desarrollado por Jesús Emanuel Rodríguez Vázquez para el artesano Luis Manuel Rodríguez Peña.

## Tecnología
HTML5, CSS3, JavaScript (ES6+). Todo local, sin dependencias externas.

## Cómo probar localmente
Abre `index.html` en un servidor local (p.ej., Live Server en VS Code) o sube los archivos a un hosting estático.

## Cómo descargar fuentes locales
1. Visita Google Fonts y descarga las fuentes "Playfair Display" (serif) y "Lato" (sans-serif) en formato .woff2.
2. Coloca los archivos en la carpeta `/fonts/`.
3. Las reglas @font-face ya están en `css/main.css`.

## Carrito y pedidos
El carrito guarda la selección en localStorage. Al finalizar se generan enlaces directos a WhatsApp y SMS.
